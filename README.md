# 🛍️ Cara — E-Commerce Website with AI Shopping Assistant

> “Imagine shopping online, but with a real-time assistant that never sleeps. That’s Cara.”

Hi, I'm **Nagarani**, and this is my take on a **modern e-commerce experience** — not just another frontend project, but a vision of how AI and UI can blend to create a **personal, intelligent, and delightful shopping journey.**

---

## ✨ What Makes Cara Different?

Cara isn’t just buttons, cards, and pages.

It’s a **thoughtfully designed virtual storefront** powered by:

- 💬 **A built-in AI chatbot** that guides users like a personal shopping assistant  
- 🛒 **Interactive shop & cart flow** — clean, minimal, and functional  
- 📱 **Fully responsive layout** that looks great on all devices  
- ⚡ **No frameworks used** — 100% handcrafted HTML, CSS, and JavaScript  
- 🎯 A UI/UX inspired by real-world brands like Zara, H&M, and Uniqlo

> **Goal:** To simulate a professional-grade product page experience without backend complexity, while giving users a taste of AI interactivity.

---

## 🌟 Core Pages & Features

| ✅ Page           | 🎯 Highlights                                    |
|------------------|--------------------------------------------------|
| `Home`           | Hero banners, featured items, clean navigation   |
| `Shop`           | Product grid, filtering logic (JS powered)       |
| `Cart`           | Add/remove items, quantity updates, total bill   |
| `Blog`           | Sample blog UI to simulate brand storytelling    |
| `Contact`        | Simple form to mimic real-world support pages    |
| **AI Chatbot**   | Friendly JS-powered assistant (non-ML version)   |

---

## 🤖 About the AI Chatbot

This is where things get fun.

Cara comes with an **AI-style chatbot** (built in vanilla JS) that can:
- Greet users
- Answer FAQs like delivery/shipping
- Suggest products
- Simulate the feel of a customer care assistant

It’s lightweight, fast, and adds **interactive personality** to the website.
